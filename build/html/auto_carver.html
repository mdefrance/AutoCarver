class AutoCarver.AutoCarver(quantitative_features: list[str], qualitative_features: list[str], min_freq: float, *, ordinal_features: list[str] | None = None, values_orders: dict[str, AutoCarver.discretizers.utils.grouped_list.GroupedList] | None = None, max_n_mod: int = 5, min_carved_freq: float = 0, sort_by: str = 'tschuprowt', str_nan: str = '__NAN__', str_default: str = '__OTHER__', output_dtype: str = 'float', dropna: bool = True, copy: bool = False, verbose: bool = False, pretty_print: bool = False)ÔÉÅ

Automatic carving of continuous, categorical and categorical ordinal features that maximizes association with a binary target.

Modalities/values of features are carved/regrouped according to a computed specific order defined based on their types:

  • [Qualitative features] grouped based on modality target rate.

  • [Qualitative ordinal features] grouped based on specified modality order

  • [Quantitative features] grouped based on the order of their values.

Uses Tschurpow‚Äôs T to find the optimal carving (regrouping) of modalities/ values of features.

Parameters:

  • values_orders (dict, default {}) ‚Äì Dictionnary of features and list of their respective values‚Äô order. Exemple: for an age feature, values_orders could be {‚Äòage‚Äô: [‚Äò0-18‚Äô, ‚Äò18-30‚Äô, ‚Äò30-50‚Äô, ‚Äò50+‚Äô]}.

  • sort_by (str, default 'tschuprowt') ‚Äì Association measure used to find the optimal group modality combination Implemented: [‚Äòcramerv‚Äô, ‚Äòtschuprowt‚Äô]

  • max_n_mod (int, default 5) ‚Äì

    Maximum number of modalities for the carved features (excluding nan).

    • All possible combinations of less than max_n_mod groups of modalities will be tested.

    Recommandation: max_n_mod should be set from 4 (faster) to 6 (preciser).

  • keep_nans (bool, default False) ‚Äì Whether or not to group numpy.nan to other modalities/values.
